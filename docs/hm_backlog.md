- **서비스 흐름도**
    
    ## Bootstrap 단계 서비스 흐름도
    
    1. **시작하자마자 사주 메인 페이지**
    2. **사주 메인 페이지에서 유도 대사가 나오고 여기에 플레이 버튼(시작하기) 배치**
    3. **쭉 유도 메시지 나오면서 intro 최종 페이지에 도달**
    4. **대립구도를 만들면서 둘 중 한 명을 선택하도록 구성 (누구의 방식대로 풀겠습니까?)**
    5. **각자의 캐릭터 특성에 맞는 대사가 나옴.**
        
        **기존의 사주를 보는 흐름에 맞게 유도 질문을 구성(생년월일, 이름, 이메일 등등)**
        
    6. **그리고 결과 페이지에서 사주 풀이를 해주는데 무료 고객의 경우 Blur로 잠금 처리가 되어 있음.**
        
        **(여기서는 알고리즘으로 그냥 대충 혹은 규칙 기반으로 진행)**
        
    7. **광고 보기나 구독의 경우에는 Blur가 풀림 (실제 Claude API 요청은 여기서)**
        
        → 결과 페이지에 공유 버튼 배치
        **→ 1 ~ 6번까지 흘러가면 신호 체크 완료**
        
    8. **마지막 스텝에서 선택받지 않은 캐릭터가 말을 걸면서 자신이라면 이런식으로 해석해줬을 것이라는 기대를 심어줌.**
